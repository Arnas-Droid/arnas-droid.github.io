  <h3>First Game Jam</h3>
  <p>While I have attempted doing a game jam before, it didn't work out. This was the first game jam where I was able to plan, implement, and publish on time. With me only finding out about it the same day the deadline was happening and with other responsibilities the same day, I only had 5 hours to do it. Thankfully, I was able to think of an idea for the game in the early morning, which ended up being a simple dodge the enemies while they attack the player idea. While not fundamentally unique, with the limited time, it worked perfectly for the game jam.</p>
</section>
